# Git이 동작하지 않을 때 확인할 점

1. Git이 설치되어 있는지 확인  
   - 명령 프롬프트(cmd)에서 아래 명령어 입력:
     ```
     git --version
     ```
   - 버전이 나오지 않으면 [git-scm.com](https://git-scm.com/)에서 Git을 설치하세요.

2. 설치 후에도 안 되면,  
   - 명령 프롬프트를 새로 열고 다시 시도하세요.
   - 환경 변수(PATH)에 git이 등록되어 있는지 확인하세요.

3. 그래도 안 되면,  
   - "git" 명령어를 찾을 수 없다는 오류 메시지 등 구체적인 에러 내용을 알려주세요.

# 참고
- Git 설치 후에는 원하는 폴더에서 `git init` 명령어를 입력하면 됩니다.
